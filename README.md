# Kremleff БанкетХолл

Сайт ресторана на Next.js (App Router), React и TypeScript.

## Запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000`.

## Проверка production-сборки

```bash
npm run lint
npm run build
npm start
```

## Структура

- `src/app` — маршруты, layout, метаданные и глобальные стили;
- `src/components` — секции страницы и интерактивные UI-компоненты;
- `public/images` — статические изображения;
- серверные компоненты используются по умолчанию, `use client` оставлен только там, где нужен браузерный API или состояние.

## Следующий этап: управляемое меню

Рекомендуемый источник данных — headless CMS. Компонент меню должен получать типизированные данные на сервере, а не хранить блюда внутри JSX. Для небольшого ресторана подойдут Sanity или Directus; изображения, категории, цены, стоп-лист и порядок вывода редактируются в их админ-панели. После публикации CMS вызывает webhook Next.js для обновления страницы.
